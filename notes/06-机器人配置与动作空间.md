## 1. G1 机器人 DOF 拆解

### 1.1 关节树

```
Unitree G1 (29 DOF)
│
├─ 腿部 (12 DOF)
│   ├─ 左腿 (6): hip_yaw, hip_roll, hip_pitch, knee, ankle_pitch, ankle_roll
│   └─ 右腿 (6): hip_yaw, hip_roll, hip_pitch, knee, ankle_pitch, ankle_roll
│
├─ 腰部 (3 DOF)
│   └─ waist_yaw, waist_roll, waist_pitch
│
├─ 手臂 (10 DOF)
│   ├─ 左臂 (5): shoulder_pitch, shoulder_roll, shoulder_yaw, elbow, wrist_roll
│   └─ 右臂 (5): shoulder_pitch, shoulder_roll, shoulder_yaw, elbow, wrist_roll
│
└─ 腕部 (4 DOF)
    ├─ 左手 (2): wrist_pitch, wrist_yaw
    └─ 右手 (2): wrist_pitch, wrist_yaw
```

### 1.2 关节配置代码

```python
# carrybox_config.py
class control:
    # 分组刚度（N·m/rad）
    stiffness = {
        'hip_yaw': 150,      # 髋偏航
        'hip_roll': 150,     # 髋滚动
        'hip_pitch': 150,    # 髋俯仰
        'knee': 300,         # 膝（2倍刚度！）
        'ankle': 40,         # 踝（软）
        'waist_yaw': 300,
        'waist_roll': 300,
        'waist_pitch': 300,
        'shoulder': 200,
        'elbow': 100,
        'wrist': 20,         # 腕（最软）
    }

    # 分组阻尼（N·m·s/rad）
    damping = {
        'hip_yaw': 2,
        'hip_roll': 2,
        'hip_pitch': 2,
        'knee': 4,
        'ankle': 1,
        'waist_yaw': 4,
        'waist_roll': 4,
        'waist_pitch': 4,
        'shoulder': 3,
        'elbow': 1,
        'wrist': 0.5,
    }
```

**设计原则**：
- 腿膝刚度最高（300）→ 支撑体重
- 腕部刚度最低（20）→ 避免破坏被抓物体
- 阻尼跟随刚度（阻尼比约 0.01-0.02）

## 2. PD 控制器实现

### 2.1 控制模式

```python
# legged_robot_config.py
class control:
    control_type = 'P'  # 纯比例控制（阻尼由 DOF velocity 提供）
    action_scale = 0.25  # 动作缩放
    decimation = 4       # 4次PD步 = 1次策略步
```

### 2.2 力矩计算

```python
# base_task.py / carrybox.py
def _compute_torques(self, actions):
    # actions: [N, 29] — 策略输出的目标关节位置偏移

    # 1. 解算目标关节位置
    action_scaled = actions * self.cfg.control.action_scale  # ×0.25
    target_dof_pos = self.default_dof_pos + action_scaled

    # 2. 域随机化：PD 参数扰动
    effective_Kp = self.Kp * self.Kp_factors     # Kp ∈ [0.9Kp, 1.1Kp]
    effective_Kd = self.Kd * self.Kd_factors

    # 3. PD 控制律
    pos_error = target_dof_pos - self.dof_pos
    vel_error = -self.dof_vel
    torques = effective_Kp * pos_error + effective_Kd * vel_error

    # 4. 电机随机化
    torques = torques * self.motor_strength         # 强度 ±10%
    if actuation_offset:
        torques += self.actuation_offset            # 偏移 ±5%

    return torch.clip(torques, -torque_limits, torque_limits)
```

**伪代码**：
```
Algorithm: PD Control with Domain Randomization

Input: actions [N, 29]  (policy output, target offset)

1. action_scaled = actions × 0.25

2. For each joint i:
     target_i = default_angle_i + action_scaled_i
     error_i = target_i - current_pos_i
     d_error_i = -current_vel_i
     torque_i = Kp_i × rand_kp_scale × error_i
              + Kd_i × rand_kd_scale × d_error_i

3. torque = torque × rand_motor_strength [0.9, 1.1]
4. torque = torque + rand_offset × torque_limits [−5%, +5%]
5. torque = clip(torque, −torque_limits, +torque_limits)

Output: torques [N, 29]
```

## 3. 域随机化（Domain Randomization）

PhysHSI 的域随机化是 sim-to-real 的核心保障：

```python
# carrybox_config.py
class domain_rand:
    use_random = True

    # === PD 参数随机化 ===
    randomize_kp = True              # Kp ∈ [0.9, 1.1]
    randomize_kd = True              # Kd ∈ [0.9, 1.1]
    randomize_actuation_offset = True # τ_offset ∈ [−5%, +5%] τ_max
    randomize_motor_strength = True   # output ∈ [0.9, 1.1]

    # === 质量/惯量随机化 ===
    randomize_payload_mass = True    # 躯干附加质量 ∈ [−2, 5] kg
    randomize_com_displacement = True # 质心偏移 ∈ [−0.1, 0.1] m
    randomize_link_mass = True       # 连杆质量 ∈ [0.8, 1.2] × nominal

    # === 接触随机化 ===
    randomize_friction = True        # 摩擦 ∈ [0.1, 1.5]
    randomize_restitution = True     # 恢复系数 ∈ [0.0, 1.0]

    # === 状态随机化 ===
    randomize_initial_joint_pos = True  # 初始关节角度 ∈ [−0.1, +0.1] × nominal
    disturbance = True                  # 随机外扰 ±50N，每 8s

    # === 延迟 ===
    delay = True                     # 观测延迟 ∈ [0, 5] timesteps
    max_delay_timesteps = 5
```

## 4. 动作空间

### 4.1 动作定义

```python
# 动作 = 29 维目标关节位置偏移
actions: torch.Tensor  # [num_envs, 29]

# 动作范围
clip_actions = 100.0  # 实际上不限幅（经 PD 后会 clamp）

# 动作缩放
action_scale = 0.25
# 实际目标偏移 = action × 0.25 rad ≈ ±14.3°
```

### 4.2 策略输出预处理

```python
# 策略输出 → PD 目标的完整链路
raw_action = policy(obs)                   # [N, 29] — 网络输出
clipped_action = clip(raw_action, -100, 100)
target_offset = clipped_action × 0.25      # 目标偏移 (rad)
target_pos = default_pos + target_offset
target_pos = clip(target_pos, lower_limits, upper_limits)
torque = Kp * (target_pos - current_pos) + Kd * (-current_vel)
```

## 6. 迁移要点：从 G1 (29 DOF) 到你的平台

### 你的平台假设

```
轮式底盘 + 单机械臂
├─ 底盘: 差速/全向 (2-3 DOF)
│   └─ 左轮速度, 右轮速度
├─ 机械臂: 6-7 DOF
│   └─ shoulder_pitch, shoulder_roll, elbow, wrist_pitch, wrist_roll, wrist_yaw
└─ 夹爪: 1 DOF
    └─ gripper_open
─────────────────────────
总计: ~10 DOF
```

### 适配清单

| PhysHSI (G1) | 你的平台 |
|-------------|---------|
| 腿部 12 DOF → 行走 | 轮式差速 2 DOF → 底盘导航 |
| 腰 3 DOF | 不需要（底盘转向代替） |
| 双臂 14 DOF | 单臂 6-7 DOF |
| 腕 4 DOF | 夹爪 1 DOF |
| 动作维度 29 | 动作维度 ~10 |
| PD 控制 | 轮子用速度控制，臂用 PD/阻抗控制 |
| Kp 分组（按关节类型） | 保持分组设计（轮子/大臂/小臂/腕） |
| 域随机化 | 全部复用（摩擦、质量、延迟、Kp/Kd） |

### 关键保持的设计

1. **动作缩放**：`action_scale = 0.25` → 保持这个值，让策略输出在合理范围
2. **PD 分组刚度**：大关节高刚度，末端低刚度
3. **域随机化参数范围**：直接复用（已验证有效）
4. **观测噪声模型**：直接复用三模式（粗/精/遮挡）
